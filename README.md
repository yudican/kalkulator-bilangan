# kalkulator-bilangan
Program ini adalah kalkulator bilangan untuk konversi bilangan lain ke bilangan lain. 
fitur:
1. Biner Ke Desimal dan sebaliknya
2. Biner Ke Octal
3. Biner Ke Hexadesimal
4. Octal ke Hexadesimal
5. Octal ke Biner
6. Octal Ke Desimal
7. dan lain lain
